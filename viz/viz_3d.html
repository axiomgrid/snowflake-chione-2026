This visualization requires simulation results.
Run one of these commands first:

# 1. Run MAP-Elites research exploration
make research

# 2. Generate the 3D visualization
make viz-3d

Or directly:

python -m src.viz_3d_builder \
  --results_dir results/<RUN_DIR>

The builder reads scan_results.csv and embeds crystal images
as base64 sprites into this HTML file (~200 MB for 300 shapes).
